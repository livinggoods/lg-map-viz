LgMapViz
=========

A module to visualize data in layers on a map

## Installation

  `npm install lg-map-viz`

## Requirements
- Mapbox Api Key
- 
## Usage

### Vue JS

    import LgMapViz from 'lg-map-viz'
    
    export default {
        ...
        components: {
            LgMapViz
        },
        ...
   
    }
    
You can then go ahead and use it in a template as follows:
    
    <template>
        <LgMapViz></LgMapViz>
    </template
